# deductible-sim
Deductible simulator for auto insurance with adjustable parameters.

## About
This project models insurance claims using Poisson frequency and exponential severity models.
It compares total losses with and without deductibles. It further allows comparison between multiple frequencies and severities.

## Features
  - Adjustable frequency/severity
  - Scaleable Monte Carlo actuarial-style simulation
  - Deductable impact analysis and profit margin
